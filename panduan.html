1. Pendahuluan

Aplikasi Supervisi PM-KBC adalah aplikasi berbasis web (single-file SPA) yang dirancang untuk membantu Pengawas Madrasah dalam melaksanakan supervisi pembelajaran, kokurikuler, ekstrakurikuler, dan iklim madrasah sesuai standar Pokjawas Jawa Timur. Mendukung jenjang umum (MI/MTs/MA) dan RA (Raudhatul Athfal) dengan instrumen terpisah.

Fitur Utama

  • Input supervisi pembelajaran umum per guru (28 indikator: 10 Perencanaan, 10 Pelaksanaan, 8 Asesmen)
  • Input supervisi pembelajaran RA per guru (28 indikator: 10 Perencanaan, 10 Pelaksanaan, 8 Evaluasi)
  • Input supervisi kokurikuler (12 indikator), ekstrakurikuler (10 indikator), iklim madrasah (11 indikator) per madrasah
  • Cetak instrumen, rekap hasil, dan laporan lengkap (cover, kata pengantar, BAB I-V, lampiran)
  • Laporan per madrasah dan per KKMA
  • Backup & restore data berbasis JSON
  • Multi-user dengan sistem login
Catatan: Semua data tersimpan di localStorage browser. Data hanya tersedia di browser dan perangkat yang sama. Lakukan backup berkala untuk mengamankan data.

3. Login dan Registrasi

Login

Langkah 1: Buka aplikasi di browser Anda.
Langkah 2: Masukkan username dan password.
Langkah 3: Klik tombol Login.

Akun Default

UsernamePassword
adminadmin123
Penting: Segera ubah password default setelah login pertama melalui menu Ubah Password.

Registrasi Akun Baru

Langkah 1: Di halaman login, klik link Daftar.
Langkah 2: Isi nama lengkap, username (min. 3 karakter), dan password (min. 4 karakter).
Langkah 3: Konfirmasi password, lalu klik Daftar.
Langkah 4: Setelah berhasil, Anda akan diarahkan ke halaman login.

4. Dashboard

Dashboard menampilkan ringkasan data supervisi pembelajaran (jenjang umum):

  • Total Supervisi - Jumlah supervisi pembelajaran yang telah dilakukan
  • Data Madrasah - Jumlah madrasah umum terdaftar
  • Data Guru - Jumlah guru umum terdaftar

Di bawah statistik tampil Daftar Supervisi beserta persentase capaian. Anda dapat:

  • Klik item untuk melihat detail
  • Klik ikon pensil untuk mengedit
  • Klik ikon hapus untuk menghapus data

Grafik capaian ditampilkan di bagian bawah untuk membandingkan skor Perencanaan, Pelaksanaan, dan Asesmen.

Catatan RA: Statistik untuk supervisi pembelajaran RA tampil terpisah pada menu Rekap Hasil RA.

5. Data Pengawas

Menu ini digunakan untuk mengisi identitas pengawas yang akan tampil di seluruh laporan (umum maupun RA).

Langkah 1: Klik menu Data Pengawas di sidebar.
Langkah 2: Isi data: Nama Pengawas, NIP, Pangkat/Golongan, dan informasi lainnya.
Langkah 3: Klik Simpan.
Catatan: Data pengawas otomatis muncul di semua jenis laporan yang dicetak.

6. Data Madrasah (Umum)

Kelola daftar madrasah binaan jenjang MI/MTs/MA. Untuk RA gunakan menu Data Madrasah RA.

Tambah Madrasah

Langkah 1: Klik menu Data Madrasah.
Langkah 2: Klik tombol + Tambah Madrasah.
Langkah 3: Isi: Nama Madrasah, NPSN/NSM, Alamat, Kabupaten/Kota, Kepala Madrasah.
Langkah 4: Klik Simpan.

Import dari Excel

Langkah 1: Klik Template Excel untuk mengunduh template.
Langkah 2: Isi data madrasah sesuai format template.
Langkah 3: Klik Import Excel dan pilih file yang sudah diisi.

7. Data Guru (Umum)

Kelola daftar guru jenjang umum yang akan disupervisi. Untuk guru RA gunakan menu Data Guru RA.

Tambah Guru

Langkah 1: Klik menu Data Guru.
Langkah 2: Klik + Tambah Guru.
Langkah 3: Isi: Nama Guru, NIP, Madrasah, Mata Pelajaran, dan Kelas.
Langkah 4: Klik Simpan.

Import dari Excel

Sama seperti data madrasah, tersedia template dan import dari Excel untuk batch entry.

Catatan: Dropdown Madrasah di form guru hanya menampilkan madrasah umum.

Langkah 2: Klik + Tambah Madrasah RA.
Langkah 3: Isi: Nama RA, NSM, Alamat, Kabupaten/Kota, Kepala RA.
Langkah 4: Klik Simpan.

Data Guru RA

Langkah 1: Buka Data RA > Data Guru RA.
Langkah 2: Klik + Tambah Guru RA.
Langkah 3: Isi: Nama Guru, NIP/NIPK, RA tempat tugas, Kelompok (A/B), dll.

14. Cetak Laporan Lengkap

Setiap modul supervisi memiliki fitur cetak laporan lengkap yang menghasilkan dokumen resmi dengan struktur:

  • Cover/Halaman Judul - Judul laporan, nama pengawas, instansi
  • Kata Pengantar
  • BAB I - Pendahuluan - Latar belakang, tujuan, sasaran
  • BAB II - Landasan Hukum
  • BAB III - Pelaksanaan - Waktu, tempat, metode
  • BAB IV - Hasil - Data hasil supervisi dan analisis
  • BAB V - Penutup - Kesimpulan dan rekomendasi
  • Lampiran - Instrumen dan data pendukung
Cara Cetak:
1. Buka menu Laporan / Cetak Laporan Lengkap pada modul yang diinginkan.
2. Pastikan data pengawas dan data supervisi sudah terisi.
3. Klik tombol Cetak Laporan.
4. Gunakan Ctrl+P untuk mencetak atau menyimpan sebagai PDF.
Catatan: Modul Supervisi Pembelajaran RA punya menu Laporan RA sendiri yang terpisah dari laporan jenjang umum.

15. Laporan Per Madrasah

Tersedia untuk modul Kokurikuler, Ekstrakurikuler, dan Iklim Madrasah. Memungkinkan cetak laporan khusus untuk satu madrasah (umum maupun RA).

Langkah 1: Buka menu Laporan Per Madrasah pada modul yang diinginkan.
Langkah 2: Pilih madrasah dari dropdown (gabungan umum + RA).
Langkah 3: Klik Cetak atau gunakan Ctrl+P.

16. Periode Supervisi

Atur Tahun Ajaran dan Semester yang akan tampil di seluruh cover laporan, kata pengantar, dan BAB Pelaksanaan. Default: Tahun Ajaran 2025-2026 Semester Ganjil.

Langkah 1: Buka menu Periode di sidebar (kelompok PENGATURAN).
Langkah 2: Isi Tahun Ajaran (mis. 2025-2026, 2026-2027).
Langkah 3: Pilih Semester (Ganjil/Genap).
Langkah 4: Klik Simpan Periode.
Catatan: Perubahan periode langsung berlaku untuk semua jenis laporan (umum, RA, kokurikuler, ekstrakurikuler, iklim madrasah, per madrasah, dan per KKMA). Tidak perlu input ulang data supervisi.

17. Ubah Password

Langkah 1: Buka menu Ubah Password di sidebar.
Langkah 2: Masukkan password lama.
Langkah 3: Masukkan password baru (min. 4 karakter).
Langkah 4: Konfirmasi password baru.
Langkah 5: Klik Simpan Password.

18. Export/Backup dan Restore

Fitur ini sangat penting untuk mengamankan data karena semua data tersimpan di browser (localStorage). Backup mencakup seluruh data: madrasah umum, madrasah RA, guru, guru RA, supervisi pembelajaran, supervisi RA, kokurikuler, ekstrakurikuler, dan iklim madrasah.

Backup Data

Langkah 1: Buka Export/Backup & Restore.
Langkah 2: Klik tombol Backup/Export Data.
Langkah 3: File JSON akan terunduh otomatis. Simpan di tempat yang aman.

Restore Data

Langkah 1: Buka Export/Backup & Restore.
Langkah 2: Klik tombol Restore/Import Data.
Langkah 3: Pilih file backup JSON yang pernah disimpan.
Langkah 4: Konfirmasi restore. Data lama akan ditimpa.
Perhatian: Restore akan menimpa SEMUA data yang ada (umum maupun RA). Backup data terbaru sebelum melakukan restore.

19. Tips dan Catatan Penting

Penyimpanan Data

  • Data tersimpan di localStorage browser, bukan di server
  • Setiap modul memakai key terpisah (mis. supervisi_data_v1, supervisi_pembelajaran_ra_v1, supervisi_kokurikuler, dll)
  • Jika data browser/cache dihapus, data supervisi akan hilang
  • Data hanya bisa diakses dari browser dan perangkat yang sama
  • Lakukan backup rutin minimal seminggu sekali

Kompatibilitas Browser

  • Gunakan browser modern: Chrome, Edge, Firefox, atau Safari versi terbaru
  • JavaScript harus aktif
  • Aplikasi responsif, bisa diakses dari HP/tablet

Tips Penggunaan

  • Isi Data Pengawas terlebih dahulu sebelum mencetak laporan
  • Daftarkan madrasah (umum/RA) sebelum input supervisi
  • Gunakan import Excel untuk input data madrasah/guru jumlah banyak
  • Klik teks indikator (biru) untuk melihat panduan penggalian data
  • Gunakan tombol Reset Skor jika ingin mengulang penilaian
  • Untuk PDF: Ctrl+P → pilih "Save as PDF"
  • Indikator dapat diedit/disesuaikan via panel Edit Indikator di halaman input supervisi
  • Untuk supervisi RA gunakan menu khusus RA, bukan menu jenjang umum
